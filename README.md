# 지역별 카페 추천 앱

## 프로젝트 소개

지역별 카페 추천 앱은 사용자가 지역을 선택하면 해당 지역의 추천 카페 목록을 보여주고, 상세 화면에서 카페 설명, 키워드, 이미지, 지도 위치를 확인할 수 있는 iOS 앱입니다.

본 앱은 실제 서비스용 데이터가 아닌 앱 기능 구현을 확인하기 위한 샘플 카페 데이터를 기반으로 제작되었습니다.

## 주요 기능

* 지역 선택 기능
* 선택한 지역의 카페 목록 표시
* 카페별 키워드 표시
* 상세 정보 화면 이동
* 카페 이미지, 주소, 설명 표시
* MapKit을 이용한 지도 위치 표시
* Navigation Controller를 이용한 뒤로가기 기능

## 사용 기술

* Swift
* UIKit
* UIStackView
* UIPickerView
* MapKit
* JSON
* Assets.xcassets

## 데이터 구성

카페 데이터는 앱 내부의 `cafeData.json` 파일로 관리합니다.

각 카페 데이터는 다음 정보를 포함합니다.

* 카페 이름
* 지역
* 주소
* 설명
* 위도 / 경도
* 이미지 이름
* 키워드

## 화면 구성

### 메인 화면

사용자가 UIPickerView를 통해 지역을 선택하면 해당 지역의 추천 카페 목록이 표시됩니다.
각 카페 항목에는 카페 이름, 주소, 키워드, 상세 버튼이 표시됩니다.

### 상세 화면

상세 버튼을 누르면 선택한 카페의 상세 화면으로 이동합니다.
상세 화면에서는 카페 이미지, 이름, 주소, 키워드, 설명, 지도 위치를 확인할 수 있습니다.

* 디저트맛집
* 사진찍기좋은
* 조용한공부
